Public SDK for building **AAIA modules and plugins**.

## Quickstart

```bash
dotnet add package AAIA.Shared.Contracts
```

Implement `IAaiaModule`, add an `aaia-extension.json` manifest, done.

→ **[Full documentation and examples](src/AAIA.Shared.Contracts/README.md)**  
→ **[Module Template](https://github.com/illeArts/aaia-module-template)** — clone and start building
